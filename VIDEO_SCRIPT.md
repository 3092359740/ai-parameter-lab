# 5 分钟录屏解说脚本

本脚本供你参考，用于录制 5 分钟以内的项目讲解视频。请根据你的实际理解自由发挥，保持自然即可。

**建议工具**：OBS Studio / 腾讯会议 / 系统自带录屏（Win+G）
**建议分辨率**：1920×1080
**语速**：正常语速，约 800-1000 字/5 分钟

---

## 开场（30 秒）

> 大家好，我是 [姓名]，学号 [学号]。今天我要展示的项目是 **AI Parameter Lab（大模型参数调优实验室）**。这是一个结合前沿 AI 技术的前端项目，目标是帮助大家直观地理解大语言模型核心参数对生成结果的影响。

---

## 项目背景与选题（45 秒）

> 为什么选择这个项目？
> 当前大语言模型非常火热，但很多同学只知道输入 Prompt，却不了解 Temperature、Top-p、Max Tokens 等参数到底在控制什么。
> 本项目把抽象的参数调优过程变成可视化、可对比的实验平台，既能学习 Prompt Engineering，也能理解模型生成行为的可解释性。

---

## 技术栈与架构（45 秒）

> 项目采用 React 18 + Vite 5 + Tailwind CSS 3 构建，使用 Lucide React 图标库。
> 支持本地模拟 LLM，无需 API Key 即可演示；同时也预留了 OpenAI GPT-4 / GPT-3.5 和 Anthropic Claude 的真实接口，只需配置环境变量即可切换。
> 部署方式灵活：Docker、Docker Compose、GitHub Pages 自动化部署都支持。

---

## 界面功能演示（2 分钟）

> 首先看页面顶部，这里显示了我的姓名和学号，点击编辑可以修改，数据会保存在浏览器本地。
>
> 左侧是模型选择与参数面板。我先选择「本地模拟 LLM」，然后调整参数。
> 比如我把 Temperature 从 0.7 调到 1.2，Top-p 调到 1.0，再输入一个提示词：「写一首关于人工智能的诗」。
> 可以看到生成的结果比较发散、有创意。
>
> 接下来我把 Temperature 调回 0.2，Top-p 调回 0.1，输入同样的提示词。
> 这次生成的结果更保守、更确定。我把两次结果都加入对比面板。
>
> 切换到「参数对比」标签，可以并排看到两组参数下的输出差异，非常直观。

## 录制提示

1. 录制前先运行 `npm run dev`，确保界面正常加载
2. 提前在顶部编辑好你的真实姓名和学号
3. 多演示几次参数调整，让对比效果更明显
4. 讲解时鼠标跟随语音，突出重点区域
5. 视频时长控制在 5 分钟以内，建议 4 分 30 秒左右
